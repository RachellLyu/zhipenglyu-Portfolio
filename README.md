# RachellLyu/zhipenglyu - Portfolio

## 项目简介
这是一个个人作品集项目，旨在展示个人的创意和专业技能。

## 项目结构
- `index.html`：项目的主页面。
- `assets`：包含项目所需的各类资源。
- `images`：存放相关图片。
- `README.md`：本说明文件。
- `elements.html`：页面元素相关代码。
- `generic.html`：通用页面代码。

## 技术栈
- CSS：用于页面的样式设计。
- SCSS：增强的 CSS 预处理器。
- HTML：构建页面结构。
- JavaScript：实现页面的交互功能。

## 如何运行
1. 克隆本项目到本地。
2. 在本地环境中打开相关文件进行查看和编辑。
